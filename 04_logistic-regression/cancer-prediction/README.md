# 癌症预测案例

## 项目简介
这是学习逻辑回归中的一个小案例，皆在掌握逻辑回归的基本应用。

## 使用技术
- 首先是用到了癌症预测的数据集文件breast-cancer-wisconsin.csv
- 导入了pandas工具，作用是读取数据
- 导入了numpy工具，作用是把异常数据处理为缺失值
- 导入train_test_split把数据集切分为训练集和测试集
- 导入StandardScaler来做标准化的数据预处理
- 导入LogisticRegression来完成逻辑回归的建模

## 项目功能
- 读取数据集，并查看数据集基本信息
- 对出现的异常值进行处理
- 提取特征和标签，随后把数据集划分成训练集和测试集
- 进行数据标准化，避免部分数据数值范围影响模型
- 进行建模并训练
- 基于训练后的模型做出评估

## 运行结果和后续的改进方案
运行结果示例：
```
y的预测值为: [4 4 2 2 4 2 2 4 2 2 ...]
准确率为: 0.9854
```
改进方案：
- 癌症预测不能仅仅以准确率为唯一评估标准，所以后期功能需使用混淆矩阵等评估方法优化
