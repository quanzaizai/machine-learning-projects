# 02 KNN

本章对应 Obsidian 笔记：`KNN算法.md`。

## 本章作用

这一章用于理解 KNN 的基本思想、距离度量、特征预处理、K 值选择、交叉验证和网格搜索。

## 当前案例

| 项目 | 练习重点 |
| --- | --- |
| `iris-knn-classifier` | 鸢尾花分类、标准化、KNN 基础流程 |
| `digits-knn-classifier` | 手写数字识别、归一化、网格搜索、模型评估 |

## 复习顺序

1. 先看 `iris-knn-classifier`，理解 KNN 分类的完整流程。
2. 再看 `digits-knn-classifier`，练习更高维数据、调参和评估。
