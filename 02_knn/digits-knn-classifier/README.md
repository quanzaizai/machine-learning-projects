# digits-knn-classifier

## 项目说明
这是一个用python完成的手写数字识别项目，导入了机器学习库的datasets数据集，本项目使用 scikit-learn 提供的数据集、数据划分、预处理、模型训练、网格搜索和评估工具完成手写数字分类任务。

## 当前功能
- 导入datasets数据集
- 基于数据集再做划分，划分成训练集和测试集
- 数据的归一化操作
- 利用交叉验证和网格搜索，在1到11之间找到最优超参数和最优评分
- 模型的评估和预测

## 核心流程
```
加载数据集 → 提取特征和标签 → 划分训练集和测试集 → 特征归一化 → 网格搜索与交叉验证 → KNN 模型训练 → 模型预测 → 模型评估
```

## 使用的工具
- 使用 sklearn.datasets.load_digits() 加载 scikit-learn 内置手写数字数据集
- sklearn.model_selection模块的train_test_split（划分训练集和测试集）,GridSearchCV（交叉验证和网格搜索）
- sklearn.neighbors模块的KNeighborsClassifier（创建KNN模型）
- sklearn.metrics模块的accuracy_score（模型的评估）
- sklearn.preprocessing模块的MinMaxScaler（数据的归一化处理）

## 数据集说明

本项目使用scikit-learn内置的`load_digits`手写数字数据集，该数据集中的每个样本是一张 8×8 的手写数字图片。图片会被展开成 64 个像素特征，用于模型输入。

- `digits.data`：特征数据，每一行表示一张图片的 64 个像素特征
- `digits.target`：标签数据，表示每张图片对应的真实数字，范围是 0 到 9
- `digits.images`：原始 8×8 图片矩阵，可用于可视化
